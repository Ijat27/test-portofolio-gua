## ✨ Features

- 🌙 **Dark / Light Mode** — Toggle tema dengan preferensi tersimpan di localStorage
- 📱 **Hamburger Menu** — Navigasi mobile dengan animasi slide & tombol ✕
- 🖱 **Smooth Scrolling** — Transisi halus antar section
- 🎯 **Active Nav Link** — Highlight otomatis link sesuai section yang aktif
- 📊 **Skill Bar Animation** — Bar animasi saat di-scroll ke section Skills
- 🎞 **Scroll Reveal** — Elemen muncul dengan animasi saat masuk viewport
- ✉ **Form Validation** — Validasi input dengan pesan error inline
- 📸 **Profile Photo** — Foto profil dengan frame dan badge "Available for work"
